Как запустить:

1. Скачать и установить node.js: https://nodejs.org/en/
Чтобы windows корректно работал с npm из консоли, рекомендуется перезагрузить компьютер 
2. Открыть проект в Visual Studio, запускать через IIS Express. 
Может быть проблема с портами, как их пофиксить, я не знаю \_(ツ)_/¯
