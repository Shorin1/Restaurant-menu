Как запустить:

1. Скачать и установить node.js: https://nodejs.org/en/
2. Перейти в папку ClientApp, открыть консоль и прописать в ней: npm install
2.1 Если windows не находит npm, то нужно перезагрузить компьютер
3. После того, как скачаются все пакеты, можно просто запускать проект через IIS Express
